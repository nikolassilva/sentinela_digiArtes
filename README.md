# sentinela_digiArtes
Este repositório se trata de um site desenvolvido por mim como trabalho final da disciplina de Programação Web I, pelo 5° período do curso de Bacharelado em Sistemas de Informação. O site tem como temática uma escola de artes digitais fictícia, chamada "Sentinela Artes Digitais". Foram utilizadas as linguagens HTML, CSS e Javascript para o desenvolvimento, aplicando também a responsividade através do CSS.
